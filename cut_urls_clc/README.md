# Обрезка ссылок с помощью CLC

Скрипт для управления короткими ссылками через API сервиса [clc.li](https://clc.li). Позволяет сокращать длинные URL и получать статистику переходов по уже существующим коротким ссылкам.

### Как установить

Для запуска скрипта клонируйте репозиторий или скачайте скрипт.
Зарегистрируйтесь на сайте и получите свой API-токен сервиса по ссылке [clc.li/API](https://clc.li/user/developers/apikeys).

Для сокрытия чувствительных данных, таких как API-токен сервиса, сохраните их в от в отдельном `.env` файле в формате `CLC_API_TOKEN=ваш_api_токен`.

Python3 должен быть уже установлен. 
Затем используйте `pip` (или `pip3`, есть конфликт с Python2) для установки зависимостей:
```
pip install -r requirements.txt
```
